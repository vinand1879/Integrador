Afim

Definição:

   Chama-se função polinomial do 1º grau, ou função afim, a qualquer função f de IR em IR dada por uma lei da forma f(x) = ax + b, onde a e b são números reais dados e a0. Na função f(x) = ax + b, o número a é chamado de coeficiente de x e o número b é chamado termo constante.

Veja alguns exemplos de funções polinomiais do 1º grau:

f(x) = 5x - 3, onde a = 5 e b = - 3
f(x) = -2x - 7, onde a = -2 e b = - 7
f(x) = 11x, onde a = 11 e b = 0


Grafico:

    O gráfico de uma função polinomial do 1º grau, y = ax + b, com a0, é uma reta oblíqua aos eixos Ox e Oy.

Exemplo:    Vamos construir o gráfico da função y = 3x - 1:

   Como o gráfico é uma reta, basta obter dois de seus pontos e ligá-los com o auxílio de uma régua:

a) Para x = 0, temos y = 3 · 0 - 1 = -1; portanto, um ponto é (0, -1).

Para y = 0, temos 0 = 3x - 1; portanto, e outro ponto é .no plano cartesiano e ligamos os dois com uma reta

0 -1
1/3 0

    Já vimos que o gráfico da função afim y = ax + b é uma reta.

O coeficiente de x, a, é chamado coeficiente angular da reta e, como veremos adiante, a está ligado à inclinação da reta em relação ao eixo Ox.

O termo constante, b, é chamado coeficiente linear da reta. Para x = 0, temos y = a · 0 + b = b. Assim, o coeficiente linear é a ordenada do ponto em que a reta corta o eixo Oy.

Exemplos comentados:

Um táxi começa uma corrida com o taxímetro a R$ 4,00. Cada quilômetro rodado custa R$ 1,50. Se no final de uma corrida, o passageiro pagou R$ 37,00, a quantidade de quilômetros percorridos foi:

a:34
b:54
c:22
d:65
e:20

Resolução

A lei de formação de custo da corrida será F(x)=1,5x + 4, onde X é os Km rodados e F(x) é o preço da corrida:
então, como o enunciado já nos informou que o preço final é 37, logo F(x) será igual a 37.

Inicialmente ele diz que o taxímetro está com 4 reias, logo b = 4, o Ax será o valor de cada Km rodado vezes a quantidade de km percorridos respectivamente(a quantidade de km é x, que é o que a questão pede) logo temos:

F(x)=37 -> Ax+b=37 -> 1,5*x + 4 =37 -> 1,5x = 37-4 -> x=33/1,5
logo x= 22 letra C
